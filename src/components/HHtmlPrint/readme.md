# HHtmlPrint

二次封装于 `VueHTML2PDF`。

更多配置查看 [vue-html2pdf](https://github.com/kempsteven/vue-html2pdf) ，[jsPDF](https://rawgit.com/MrRio/jsPDF/master/docs/jsPDF.html)

## Feature

- 增加了弹窗预览
- 增加了进度通知
- 预设了一些配置

## Methods

| name        | 描述                |
| ----------- | ------------------- |
| `generatePdf` | 主要生成 pdf 的方法 |
| `open`        | 打开弹窗的方法      |
